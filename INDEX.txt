DATA STRUCTURE
================================================================================

Heatmap Points: 188 [latitude, longitude, intensity]
- Intensity range: 0.95-1.0
- Gradient colors: Navy → Teal → Orange → Red
- Real output from contextual ML model

Verified ULAB Sites: 13 objects
- Properties: id, name, lat, lng
- Represented as purple circles
- Ground truth from Pure Earth TSIP

Detected ULAB Sites: 25 objects
- Properties: id, lat, lng, riskScore, status, dateDetected, flaggedBecause
- Risk scores: 0.84-0.94
- Status: Pending Verification, Scheduled for Visit, Under Review
- Each has detailed explanation of flagging reason

Detected Smelter Sites: 8 objects
- Similar structure to ULAB sites
- Risk scores: 0.75-0.94
- Explanations reference satellite features

Feature Importance: Arrays of 8 features each
- ULAB features: Battery shops (0.95) → Transportation (0.60)
- Smelter features: Thermal (0.98) → Transportation (0.55)
- Used for explaining model decisions
